# ClearBlueSky Stock Scanner v7.7

See the main **[README.md](../README.md)** in the project root for full documentation.

## Quick Reference

- **4 Scanners:** Trend (long-term), Swing (emotional dips), Watchlist, Pre-Market
- **Index options:** S&P 500, ETFs
- **New in v7.7:** Ticker enrichment (earnings/news/leveraged), overnight markets, insider data in Trend & Swing, AI gives 5+ picks
- **Run:** `python app.py` (GUI) or `python scanner_cli.py --scan <type>` (CLI)

See **CLI_FOR_CLAUDE.md** for CLI automation details.
